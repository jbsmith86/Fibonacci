Fibonacci
=========
Calculates fibonacci numbers to the Nth place given.

Use main.rb to call the fibonacci class

example main.rb file:

```ruby
require './lib/fibonacci'

a = Fibonacci.new

puts a.calculate(5)
puts a.calculate(9)
puts a.calculate(17)
puts a.calculate(512)
```
